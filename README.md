# Hortech Frontend

## Pré-requistos

1. Execute o comando para instalar as dependências:
   ```sh
   yarn
   ```
2. Crie um arquivo `.env` na pasta raiz do projeto e copie as varíaveis de ambiente presentes no arquivo `.env.example` .

## Rodando a aplicação

1. Execute o comando para o ambiente de desenvolvimento:
   ```sh
   yarn start
   ```
